# Badge

用于角标等辅助提示信息的组件。

## 组件

### Badge

`<Badge>` 组件。

#### Props

##### `component`

> PropType: `node`

组件使用的元素，默认为 `span`。


##### `href`

> PropType: `string`

如果设置 `href` 属性将忽略 `component` 属性，渲染为 `<a>`。


##### `amStyle`

> PropType: `string`

颜色样式。可选值：

- `'primary'`
- `'secondary'`
- `'success'`
- `'warning'`
- `'alert'`
- `'dark'`


##### `rounded`

> PropType: `bool`

是否应用椭圆样式。

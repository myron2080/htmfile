# Utility

常用样式 class 封装。

## 布局相关

### `.root`

```css
.root {
  display: block;
  height: 100%;
  width: 100%;
  overflow: hidden;
}
```

### float

- `.cf`: 清除浮动
- `.fl`: 向左浮动
- `.fr`: 向右浮动
- `.fn`: 不浮动


## 文本

### 单行文字截断

- `.text-truncate`: 需要添加 `block` 或 `inline-block` 样式。

### 文本对齐

- `.text-left`: 左对齐
- `.text-center`: 居中
- `.text-right`: 右对齐
- `.text-justify`: 两端对齐
